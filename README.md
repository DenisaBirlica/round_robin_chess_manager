# Checkmate Manager

A professional round-robin chess tournament engine.

## Features
- **Live Standings**: Real-time tournament leaderboard.
- **Round-Robin Engine**: Automated pairing and scheduling.
- **Cloud Sync**: Data is saved securely in the cloud.
- **Cross-tables**: Visual representation of tournament progress.

## How to use
1. Add players to your tournament.
2. Generate the rounds.
3. Enter results as games are finished.
4. Watch the standings update automatically!

## Deployment
This project is set up to be deployed easily. No special configuration is required for standard use.
